
# 📝 Anotaciones del Proyecto - Sistema de Pasajes

## 📌 Recordatorios Críticos
- **Estructuras permitidas:** Solo listas y listas anidadas. Prohibido diccionarios para lógica central.
- **Sin POO:** Toda la lógica con funciones y programación estructurada.
- **Validación previa:** No usar `try/except`. Validar con RegEx antes de operar.

## 🔧 Controles Importantes
- [ ] Botón de salida en selección de asiento (ingreso `'0'`)
- [ ] Validar límites de matriz antes de acceder a índices
- [ ] Manejar lista vacía en `reduce()` para recaudación
- [ ] Verificar que `filter()` encuentra ID antes de iterar resultado

## ⚠️ Errores Comunes a Evitar
- `map()` solo para calcular precio final (precio base + 16%)
- `reduce()` solo para sumar `ventas_diarias`
- `filter()` solo para buscar viajes
- No codificar lógica con diccionarios en módulos centrales
